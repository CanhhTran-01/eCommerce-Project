
# Ecommerce Backend (ecommerce-be)

A simple **Spring Boot backend project** for an ecommerce website, built mainly for **learning and internship application purposes**.

---

## Tech Stack

- **Java**: 21
- **Spring Boot**: 3.x
- **Build Tool**: Maven
- **Spring Data JPA**
- **RESTful API**
- **Database**: MySQL

---

## Project Structure
````
com.myproject.ecommerce
├── controller
├── dto
     ├── request
     └── response
├── entity
├── enums
├── exception
├── mapper
├── repository
├── service
└── utils
````

---

The project currently provides basic CRUD APIs for core entities.
Detailed API documentation will be added later.



##  Run the Application
```bash
 ./mvnw spring-boot:run
```
